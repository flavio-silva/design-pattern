#Passos para instalação do site
1. O servidor do php tem de executar na pasta public onde é o document root da aplicação.
2. As Classes e Interface estão dentro da pasta src
3. No index.php existe um exemplo de uso do formulário orientado a objeto
4. A renderização acontece na view contato, dentro da pasta view
5. Este projeto depende do Aura\Di, necessário para funcionamento dos exemplos. Obs.: Será necessário instalar via composer
6. Dependency Injection Container está instalado na pasta vendor via composer (Aura\DI)
7. O DIC está configurado no index
